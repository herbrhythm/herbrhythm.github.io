关于Herbrhythm开放源代码项目

Herbrhythm通过开放源代码计划将非敏感业务代码进行开源和推广的计划。

希望这些项目能帮助到更多有需要的人，并获得反馈，改进和建议。

Herbwebdav

Herbwebdavd是一款轻量级的Webdav服务。

主要针对的使用场景为

  • 虚拟机里宿主机与客户机交换文件
  • 内网小范围传文件，作为文件中心使用
  • 临时需要上传文件到服务/VPS上，通用http端口，不需要额外安装服务和调整防火墙
GithubCoding Herbbuckets

Herbbuckets是一款文件上传服务

  • 支持本地文件系统和远程系统多种驱动方式
  • 支持Bucket级别的权限管理
  • 支持有过期失效的链接
